# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [Unreleased]

### Added
- Initial open source release of Devometric Community Edition
- Support for multiple LLM providers (Ollama, OpenAI, Anthropic)
- Embeddable chat widget with customizable branding
- Organization configuration (system prompts, policies, coding standards)
- Multi-language support (English, Finnish, Swedish, Spanish, German)
- Self-hosted deployment with Docker Compose
- Airgapped deployment support with Ollama

### Changed
- Rebranded from Stackpact to Devometric
- Simplified routing for Community Edition
- Developer-friendly welcome page

## [1.0.0] - 2024-01-03

### Added
- Initial release
- Rails 8 API backend
- PostgreSQL database
- Redis for caching
- Hotwire for real-time streaming
- TailwindCSS styling
- Admin API endpoints
- Embed API for widgets
- Domain whitelisting
- Usage analytics
