Test and customize your AI assistant widget

Quick Start

  1. Make sure the Rails server is running: bin/dev
  2. Open the browser console (F12) to see debug logs
  3. Click the chat button in the bottom-right corner
  4. Try sending a message to test the AI response
Tip: Configure your LLM provider in .env file. Supports Ollama (local), OpenAI, and Anthropic.

Embed Code

Add this script to any page to embed the widget:

<script
  src="http://localhost:3000/widget.js"
  data-key="YOUR_EMBED_KEY"
  data-position="bottom-right"
></script>

Configuration Options

data-key       Your organization's embed key (required)
data-position  Widget position: bottom-right, bottom-left
data-theme     Color theme: auto, light, dark
data-locale    Language: en, fi, sv, es, de
